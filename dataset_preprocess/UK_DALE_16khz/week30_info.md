# UK-DALE Dataset Description - House 2 (Week 30)

## Overview
This folder contains high-frequency Voltage and Current (VI) waveform data from the **UK-DALE** (UK Domestic Appliance-Level Electricity) dataset.

*   **House:** House 2
*   **Location:** London, UK
*   **Sampling Rate:** 16 kHz
*   **Format:** `.flac` (lossless audio compression)

## Temporal Coverage
The data in this folder represents exactly **one week** of continuous monitoring.

| :--- | :--- |
| **Start Time** | Monday, July 22, 2013, 07:00:00 |
| **End Time** | Monday, July 29, 2013, 07:00:00 |
| **Total Duration** | 7 Days (168 hours) |
| **Unix Range** | 1374447600 to 1375052400 |

## Data Structure
*   **Files:** 1-hour increments.
*   **Naming Convention:** `vi-[timestamp]_[random_id].flac`
*   **Size:** ~190 MB per file (~31 GB total for the week).
